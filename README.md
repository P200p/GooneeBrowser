# Goonee Browser

เว็บเบราว์เซอร์สำหรับ Android ที่เน้นความคล่องตัวและการปรับแต่งด้วยสคริปต์ (JavaScript Shortcuts) พร้อมฟีเจอร์แยกหน้าจอ (Split View)

## ฟีเจอร์หลัก (Key Features)

- **ระบบ Multi-tab**: รองรับการเปิดหลายหน้าเว็บพร้อมกัน จัดการแท็บได้ง่ายจากแถบด้านบน
- **โหมดแยกหน้าจอ (Split Mode)**: สามารถดูหน้าเว็บสองหน้าพร้อมกันในแนวตั้ง พร้อมแถบปรับขนาด (Split Handle) ที่ลากปรับความสูงได้ตามต้องการ
- **JavaScript Shortcuts**: ระบบรันสคริปต์ JavaScript เพื่อปรับแต่งหน้าเว็บ เช่น Dark Mode, VConsole, Eruda Console หรือ Google Translate
- **Auto-run Script**: ตั้งค่าให้สคริปต์ทำงานโดยอัตโนมัติเมื่อโหลดหน้าเว็บเสร็จ
- **แถบทางลัด (Main Shortcut Bar)**: เข้าถึงสคริปต์ที่ใช้บ่อยได้ทันทีจากหน้าจอหลัก
- **Full Screen Mode**: โหมดเต็มจอเพื่อการอ่านเนื้อหาที่เต็มตา
- **Draggable FAB**: ปุ่มเมนูที่สามารถลากย้ายตำแหน่งได้อิสระ ไม่บังเนื้อหาหน้าเว็บ

## วิธีการใช้งาน (Usage)

### การจัดการแท็บ
- **เปิดแท็บใหม่**: กดปุ่ม `+` ที่แถบด้านบน
- **สลับแท็บ**: กดที่ไอคอน Favicon ของแท็บที่ต้องการในแถบด้านบน
- **ปิดแท็บ**: กดค้าง (Long Press) ที่ไอคอนแท็บ จะมีหน้าต่างยืนยันการปิดแท็บแสดงขึ้นมา

### โหมดแยกหน้าจอ
- กดปุ่ม **Split** เพื่อเปิด/ปิดโหมดแยกหน้าจอ
- ในโหมดแยกหน้าจอ หน้าเว็บที่ถูกควบคุม (Focus) จะมีแถบ URL อัปเดตตามหน้านั้นๆ
- สามารถแตะที่หน้าเว็บใดเว็บหนึ่งเพื่อเปลี่ยน Focus ของการควบคุม
- ลากที่แถบตรงกลาง (Split Handle) เพื่อปรับสัดส่วนความสูงของทั้งสองหน้าจอ

### ระบบสคริปต์ (Shortcuts)
- กดปุ่มสายฟ้า (FAB) เพื่อเปิดแผงจัดการสคริปต์
- **ย้ายตำแหน่งปุ่ม**: สามารถแตะค้างแล้วลากปุ่ม FAB ไปยังตำแหน่งที่ต้องการได้
- **เพิ่มสคริปต์**: กดปุ่ม `+` ในหน้าแผงสคริปต์
- **จัดการสคริปต์**: กดค้างที่ชื่อสคริปต์เพื่อ แก้ไข หรือ ลบ
- **แสดงบนหน้าหลัก**: กดไอคอนรูป "ตา" เพื่อนำสคริปต์นั้นไปแสดงเป็นปุ่มทางลัดที่แถบด้านล่างหน้าจอหลัก

## การติดตั้ง (Installation)

1. Clone โปรเจกต์นี้ไปยังเครื่องของคุณ
2. เปิดโปรเจกต์ด้วย Android Studio
3. Build และติดตั้งลงบนอุปกรณ์ Android หรือ Emulator (แนะนำ Android 7.0 ขึ้นไป)

## เทคโนโลยีที่ใช้
- Kotlin
- Android Jetpack (ViewBinding, AppCompat)
- WebView with JavaScript Interface
- RecyclerView for Tabs & Shortcuts
